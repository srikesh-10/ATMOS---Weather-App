Atmos - Premium Weather App
=============================

A beautiful, modern weather application providing real-time weather updates, dynamic background images, and location-based data.

Features
--------
- **Real-Time Weather**: Accurate temperature, wind speed, and conditions powered by Open-Meteo.
- **Dynamic Backgrounds**: Stunning city wallpapers that update automatically based on the searched location (via Pollinations.ai).
- **Day/Night Cycle**:
  - UI icons adapt to day/night status.
  - Interactive Sun/Moon indicator.
  - Main weather icons change (e.g., Sun for day, Moon for night).
- **Interactive Map**: specific location visualizer using Leaflet.js.
- **Geolocation**: Automatically detects your current location on startup.
- **Search**: Find weather for any city globally.
- **Premium Design**: Glassmorphism UI, smooth animations, and responsive layout.

Technologies
------------
- HTML5 & CSS3 (Custom Properties, Flexbox/Grid, Glassmorphism)
- JavaScript (ES6+, Fetch API)
- APIs:
  - Open-Meteo (Weather Data)
  - Nominatim OpenStreetMap (Geocoding)
  - Pollinations.ai (Generative Background Images)
- Libraries:
  - Leaflet.js (Maps)
  - Google Fonts (Outfit)

How to Run
----------
1. Simply open `index.html` in any modern web browser.
   - No build step or server required!
2. Allow "Location Access" when prompted to see weather for your current area.
3. Use the search bar to find other cities.
